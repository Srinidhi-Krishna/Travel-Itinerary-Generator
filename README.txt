# AI Travel Itinerary Generator

An AI-powered travel itinerary generator that creates personalized travel plans based on user preferences, budget, destination, calendar availability, and weather conditions. The application provides an interactive, step-by-step planning experience and supports itinerary preview, saving, and booking assistance.

## Features
- Destination and budget-based trip planning  
- Personalized travel preferences collection  
- Calendar and real-time weather integration  
- AI-generated day-wise travel itineraries  
- Trip preview and saved itineraries  
- Video-based destination inspiration  
- Booking assistance for travel planning  
- Customizable user settings  

## Tech Stack
- **Python**
- **Streamlit**
- **Hugging Face APIs**
- **Unsplash API**
- **Pexels API**
- **Web scraping & API integration**
